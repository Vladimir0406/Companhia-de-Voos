# Companhia-de-Voos
  Este algoritmo simula de forma simplificada um sistema de gerenciamento de uma companhia aérea, ultilizando dos princípios conceitos de algoritmos e estruturas de dados e servindo como forma de prática dos tais.
  A aplicação permite adição de passageiros no voo ultilizando a estrutura lista e é capaz de criar uma fila para colocar passageiros em espera caso o voo esteja com todos os assentos ocupados. O algoritmo permite as seguintes funcionalidades:
- Adição de voos - Origem, destino e numero de vagas
- Adição de passageiros - Apenas o CPF
- Listagem de voos e passageiros
- Remoção de voos e passageiros
